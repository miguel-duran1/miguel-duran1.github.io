Sobre mí

Como Técnico Superior en Desarrollo de Aplicaciones Web, combino sólidos conocimientos técnicos con mi experiencia como deportista profesional, aportando disciplina, trabajo en equipo y capacidad para trabajar bajo presión. Me especializo en el desarrollo de aplicaciones web modernas utilizando tecnologías como Laravel, Vue.js y AWS para crear soluciones escalables y de alta calidad.

Mi enfoque metodológico y mi capacidad para aprender rápidamente me permiten adaptarme a nuevos entornos tecnológicos. Busco activamente incorporarme a un equipo de desarrollo donde pueda continuar creciendo profesionalmente mientras aporto valor desde el primer día con mi formación técnica y habilidades personales.

Habilidades Destacadas

Ofrezco una combinación de habilidades técnicas de desarrollo web full-stack y competencias profesionales adquiridas tanto en mi formación académica como en proyectos prácticos.

Desarrollo Front-End

Creo interfaces responsivas con HTML5, CSS3 y JavaScript, aplicando metodología Mobile First. Experiencia desarrollando SPAs con Vue.js, implementando componentes reutilizables y flujos de datos reactivos.

Back-End y API REST

Desarrollo en Laravel con arquitectura MVC y creación de APIs RESTful seguras. Implementación de autenticación con Laravel Sanctum y gestión eficiente de bases de datos. Experiencia con Docker para entornos de desarrollo consistentes.

DevOps y Cloud

Configuración y despliegue de aplicaciones en AWS EC2, implementación de HTTPS con Let's Encrypt y gestión de dominios. Experiencia en configuración de servidores web y optimización de rendimiento para producción.

Control de Versiones

Dominio de Git con metodologías de trabajo colaborativo a través de GitHub y GitLab. Experiencia en gestión de ramas, resolución de conflictos, y optimización del flujo de trabajo en equipos de desarrollo.

Soluciones Empresariales

Desarrollo de una plataforma de gestión hotelera con Laravel y Vue, implementando sistema de roles, reservas personalizadas y pruebas unitarias. Enfoque en crear soluciones adaptadas a necesidades reales de negocio.

Infraestructura Cloud

Configuración de arquitecturas en AWS para aplicaciones web, implementación de medidas de seguridad y optimización de recursos. Experiencia en configuración de instancias EC2, políticas de seguridad y gestión de dominios en la nube.
